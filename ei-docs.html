Utility Javascrpt eI() e eIs()

Creazione di un nuovo elemento

Vanilla JavaScript

const newElement = document.createElement('div', 'my class');

Con eI()

Puoi usare la stessa scrittura di createElement 
const newElement = eI('div', 'my class');
Oppure puoi Scrivere direttamente l'html 
const newElement = eI('<div class="my class">Text</div>');

Selezione di un elemento tramite ID

Vanilla JavaScript

const newElement = document.getElementById('myId');

Con eI()

const newElement = eI('#myId');

Selezione di un elemento tramite classi

Vanilla JavaScript

const newElement = document.querySelector('.jsClass');

Con eI()

const newElement = eI('.jsClass');

AppendTo

Vanilla JavaScript

el = createElement('div');
el.textContent = 'My Div';
container.appendChild(el)

Con eI()

eI('div', {text: 'My Div', to: container});
Oppure 
eI(container).eI('div', {text: 'My Div'});

Modificare un elemento esistente

Vanilla JS

const targetElement = document.getElementById('targetElement');
targetElement.textContent = 'Testo modificato!';
targetElement.style.color = 'red';

Con eI()

eI('#targetElement', {
    text: 'Testo modificato!',
    style: { color: 'red' }
});

Aggiungere un evento

Vanilla JS

const button = document.getElementById('exampleButton');
button.addEventListener('click', () => {
    alert('Hai cliccato il pulsante!');
});

Con eI()

eI('#exampleButton', {
    click: () => alert('Hai cliccato il pulsante!')
});

Sostituire un elemento esistente

Vanilla JS

const oldElement = document.getElementById('oldElement');
const newElement = document.createElement('div');
newElement.textContent = 'Nuovo elemento!';
oldElement.parentNode.replaceChild(newElement, oldElement);

Con eI()

eI('<div>Nuovo elemento!</div>', {
    replaceChild: '#oldElement'
});

Rimuovere un elemento

Vanilla JS

elementToRemove.parentNode.removeChild(elementToRemove);

Opzioni disponibili per eI()

Opzione Tipo Descrizione
to string o HTMLElement Aggiunge l'elemento come figlio di un altro elemento.
before string o HTMLElement Inserisce l'elemento prima di un altro elemento.
after string o HTMLElement Inserisce l'elemento dopo un altro elemento.
replace string o HTMLElement Sostituisce il contenuto di un elemento con l'elemento corrente.
replaceChild string o HTMLElement Sostituisce completamente un elemento con l'elemento corrente.
remove boolean Rimuove l'elemento dal DOM.
click, mouseover, ecc. function Aggiunge gestori di eventi come click, mouseover, ecc.
style object Applica stili CSS all'elemento.
class string o array Aggiunge una o pi√π classi all'elemento.
removeClass string Rimuove una classe specifica dall'elemento.
replaceClass array Sostituisce una classe con un'altra.
id string Imposta l'ID dell'elemento.
text string Imposta il testo dell'elemento.
html string Imposta l'HTML interno dell'elemento.

Uso di eI come funzione del DOM

Gli elementi del DOM che sono richiamati da eI acquistano due metodi eI e eIs.
eI(el1).eI(el2). L'el2 viene inserito dentro el1 tramite appendChild. La funzione ritorna l'el2.

Vanilla JavaScript

// Creare un div con classi e aggiungerlo a un contenitore
const newElement = document.createElement('div');
newElement.classList.add('alert', 'alert-info');
newElement.textContent = 'Elemento creato con createElement';
document.getElementById('createElement').appendChild(newElement);

Con eI()

// Creare un div con classi e aggiungerlo a un contenitore
el = eI('#createElement').eI('div', 'alert alert-info');
el.textContent = 'Elemento creato con createElement';

Con eI()

eI('#dynamicElement', { text: 'Elemento creato con createElement' });

4. Utilizzo di eIs() con un selettore e una funzione

Vanilla JavaScript

// Applicare una funzione a tutti gli elementi <li> all'interno di un <ul>
const listItems = document.querySelectorAll('ul li');
listItems.forEach((el, i) => {
    el.classList.add('list-item');
    el.textContent = `Elemento ${i + 1}`;
});

Con eIs()

// Applicare una funzione a tutti gli elementi <li> all'interno di un <ul>
eIs('ul li', (el, i) => {
    el.classList.add('list-item');
    el.textContent = `Elemento ${i + 1}`;
});

5. Utilizzo di eIs() con un NodeList e una funzione

Vanilla JavaScript

// Ottenere un NodeList di tutti gli elementi con la classe .myClass
const nodeList = document.querySelectorAll('.myClass');

// Applicare una funzione a tutti gli elementi nel NodeList
nodeList.forEach((el, i) => {
    el.classList.add('highlight');
    el.textContent = `Elemento ${i + 1} evidenziato`;
});

Con eIs()

// Ottenere un NodeList di tutti gli elementi con la classe .myClass
const nodeList = document.querySelectorAll('.myClass');

// Applicare una funzione a tutti gli elementi nel NodeList
eIs(nodeList, (el, i) => {
    el.classList.add('highlight');
    el.textContent = `Elemento ${i + 1} evidenziato`;
});

8. Aggiungere un elemento con animazione

Vanilla JS

document.getElementById('addButton').addEventListener('click', () => {
    const newElement = document.createElement('div');
    newElement.textContent = 'Elemento ' + counter;
    newElement.classList.add('box', 'fade-in');
    document.getElementById('container').appendChild(newElement);
    counter++;
});

Con eI()

let counter = 1;
eI('#addButton', {
    click: () => {
        eI('<div class="box fade-in">Elemento ' + counter + '</div>', {
            to: '#container'
        });
        counter++;
    }
});
