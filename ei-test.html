Test di tutti gli esempi di eI.js e eIs.js

1. Creazione di un nuovo elemento

eI('<div>Ciao, sono un nuovo elemento!</div>', { to: '#example1' });
Output atteso: Un nuovo div con il testo "Ciao, sono un nuovo elemento!" dovrebbe apparire nel box sottostante.

2. Selezione di un elemento tramite ID

eI('#example2', { text: 'Testo modificato!' });
Output atteso: Il testo dell'elemento con ID "example2" dovrebbe cambiare in "Testo modificato!".
Testo originale

3. Selezione di un elemento tramite classi

eI('.example3', { text: 'Testo modificato!' });
Output atteso: Il testo dell'elemento con classe "example3" dovrebbe cambiare in "Testo modificato!".
Testo originale

4. AppendTo

eI('div', { text: 'Elemento aggiunto', to: '#example4' });
Output atteso: Un nuovo div con il testo "Elemento aggiunto" dovrebbe apparire nel box sottostante.

5. Modificare un elemento esistente

eI('#example5', { text: 'Testo modificato!', style: { color: 'red' } });
Output atteso: Il testo dell'elemento con ID "example5" dovrebbe cambiare in "Testo modificato!" e il colore del testo dovrebbe diventare rosso.
Testo originale

6. Aggiungere un evento

eI('#example6Button', { click: () => alert('Hai cliccato il pulsante!') });
Output atteso: Cliccando il pulsante, dovrebbe apparire un alert con il messaggio "Hai cliccato il pulsante!".

7. Sostituire un elemento esistente

eI('<div>Nuovo elemento!</div>', { replaceChild: '#example7' });
Output atteso: L'elemento con ID "example7" dovrebbe essere sostituito con un nuovo div contenente il testo "Nuovo elemento!".
Elemento originale

8. Aggiungere stili e classi

eI('#example8', { class: 'text-success fw-bold', style: { fontSize: '20px' } });
Output atteso: L'elemento con ID "example8" dovrebbe avere il testo in grassetto, di colore verde e con dimensione del font di 20px.
Testo originale

9. Aggiungere un elemento con più attributi

eI('<div data-custom="value">Elemento con attributi personalizzati</div>', { to: '#example9' });
Output atteso: Un nuovo div con il testo "Elemento con attributi personalizzati" e l'attributo "data-custom" dovrebbe apparire nel box sottostante.

10. Rimuovere un elemento

eI('#example10', { remove: true });
Output atteso: L'elemento con ID "example10" dovrebbe essere rimosso dal DOM.
Questo elemento verrà rimosso

11. Applicare una funzione a tutti gli elementi di una lista

eIs('ul li', (el, i) => {
    el.classList.add('list-item');
    el.textContent = `Elemento ${i + 1}`;
});
Output atteso: Tutti gli elementi <li> all'interno di <ul> dovrebbero avere la classe "list-item" e il testo modificato in "Elemento 1", "Elemento 2", ecc.

12. Applicare stili a tutti gli elementi con una classe specifica

eIs('.example12', (el) => {
    el.style.color = 'blue';
    el.style.fontWeight = 'bold';
});
Output atteso: Tutti gli elementi con la classe "example12" dovrebbero avere il testo blu e in grassetto.
Testo originale
Testo originale

13. Aggiungere un evento a tutti i pulsanti

eIs('button', (el) => {
    el.addEventListener('click', () => {
        alert(`Hai cliccato il pulsante: ${el.textContent}`);
    });
});
Output atteso: Cliccando su qualsiasi pulsante, dovrebbe apparire un alert con il testo del pulsante cliccato.
